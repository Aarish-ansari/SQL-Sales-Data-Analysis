# SQL-Sales-Data-Analysis

## Project Overview
This project analyzes sales data using MySQL.

## Dataset
- Customer Name
- Product
- City
- Quantity
- Amount
- Payment Mode
- Order Date

## Business Questions Solved
- Month-on-Month Revenue Growth
- Quarter-wise Revenue
- Top Selling Products
- Revenue Contribution by City
- Average Order Value
- Repeat Customers
- Top Customers
- Payment Mode Analysis
- Top 3 Products in Each City
- Best Selling Product by City

## Skills Used
- GROUP BY
- HAVING
- CASE
- CTE
- Window Functions
- RANK()
- DENSE_RANK()
- LAG()
- Subqueries
- Aggregate Functions
